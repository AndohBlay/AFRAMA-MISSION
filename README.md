Aframa Mission School Website
A simple school website that works on all devices.

What's Included
index.html - Main website page

main.css - All the styles

script.js - Interactive features

Images/ - Pictures for the website

What It Does
Shows school information

Displays management team

Has a contact form

Works on phones, tablets, and computers

How to Use
Open index.html in any web browser

That's it! No installation needed

What You Can Change
Text: Edit words in index.html

Pictures: Replace images in Images/ folder

Colors: Change colors in main.css

Form: Update form in script.js

Pages You'll See
Home - Welcome message

Management - School staff

About - School information

Contact - Message form

On Mobile Phones
Menu becomes a button (≡)

Text sizes adjust

Forms fit the screen

Easy to tap buttons

Colors Used
Blue: School color

White: Background

Black: Text

Yellow: Highlights

Things That Move
Buttons change color when hovered

Pages slide in when scrolling

Mobile menu slides down

Form checks your input

Tips
Keep images small (under 500KB)

Test on your phone

Update contact info

Add real school content

Need Help?
Check if all files are in same folder

Make sure image names match exactly

Test in different browsers

Look for error messages in browser console (F12)
